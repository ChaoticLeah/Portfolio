The Devils Dungeon

The Devils Dungeon is a project I made for the Devtober 2020 game jam. I learned to work under short time constraints. In the end I was able to ship a full grid based puzzle game.

My Role & Things I did

  • Programmed a rendering system
  • Programmed all the logic for the game
  • Rapidly prototyped game mechanics and tested them
  • Did the artwork for the game
  • Made a level editor

Software Used

  • HTML
  • Javascript
  • Github
  • Aseprite
  • SFXR
