The Orbetorian

The Orbetorian is a sandbox game where everything is procedurally generated. Its like terraria and no mans sky combined. This was a project that took around 1-2 years. I worked on this in Java with a friend.

My Role & Things I did

  • Made the UI and its framework
  • Made a text renderer
  • Added Discord rich presence
  • Bug fixes
  • Keybinds
  • Made the save file self fixing if corrupted
  • Planet Pinning
  • Made many of the particles in the game
  • Fancy grass that reacts to the enviorment
  • Quality of life inventory and mining/building features
  • Debug tools and preformance monitoring
  • Designed some sfx (Door open/close, and walking sound)
  • Recipe book
  • Auto updating
  • Some textures

Software Used

  • Java
  • Github
  • Aseprite
  • Audacity (For sound mixing)

Unique challanges

This project was made in a custom game engine so we had to make everything ourselves. Also when doing textures we had to do most things in black and white since color was added at runtime based on the materials it was made out of.
