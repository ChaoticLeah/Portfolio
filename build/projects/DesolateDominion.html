Desolate Dominion

Desolate Dominion is a game I made on Roblox to learn more about procedural terrain generation. Throughout this whole process I had a lot of fun coming up with ways to generate intresting and accurate terrain, and weather all on very limited server processing power in lua.

My Role & Things I did

  • Programmed a procedural terrain system allowing for biomes, objects, and more based on things like heat maps, errosion maps, and more!
  • Programmed a custom anticheat
  • Did the artwork and modeling for the game
  • Programmed a dynamic weather system
  • Programmed a chunk system
  • Programmed a building and mining system.
  • Made the terrain system simple and extendable.

Software Used

  • Lua
  • Roblox
  • Rojo & Aftman
  • Blender
