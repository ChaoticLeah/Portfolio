Forkey

Forkey is a project based on Misskey.io which has many accessability, moderation, and other improvements. It was created based on what I learned from Sharkey. Sharkey is based on Misskey which has its own issues and isnt as well tested especially for big instances. That is why we chose Misskey.io to base it off of since its running one of the biggest misskey instances.

Forkey takes a lot of the improvements I did on Sharkey and also implements some of its own features for instance customization, digital wellbeing, and more.

My Role & Things I did

  • All the Main Features You Love from Other Forks
    • Account registration approval
    • Notification dot in favicon
    • Firefish mfm
    • Friendly CAPTCHA
    • Number lines in code blocks
    • Splitting isCat and cat speak
    • ListenBrainz integration
    • Wildcard antennas
    • Mute, blocked, and other profile badges
    • Pickable MFM search engine
    • Force poll refreshing
  • Accessibility Features
    • Warning when you forget to add alt text to a post
    • Search drive by alt text
    • Incoming media under a CW can now be marked as sensitive automatically
    • Digital wellbeing features, including the ability to hide follower, following, or note count stats
    • Fixed blocks
    • Posts per day for users shown
    • More mobile-friendly UI
    • Ability to reorder poll options
  • Better Onboarding
    • Enhanced onboarding experience for new users
  • More Customization
    • Instance customization:
      • Donate to instance button
      • Custom email templates for all outgoing emails
  • Better Moderation Tools & Performance
    • Moved delete under post dropdown to an admin submenu to avoid accidental deletions
    • Allow admins to force a user CW
  • Additional Improvements
    • Ability to hide home posts from RSS feeds and profile JSON
